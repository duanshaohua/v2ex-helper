# V2EX HELPER

## 使用说明

1. 下载最新的 `v2ex-helper.crx` 文件（[下载点这里](https://github.com/hanzichi/v2ex-helper/releases)）
2. 打开 chrome 浏览器，地址栏输入 `chrome://extensions/`
3. 将下载的 `v2ex-helper.crx` 文件拖入打开的页面即可进行安装
